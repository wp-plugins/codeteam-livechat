== Description ==

This is a chat plugin using which admin users can chat with website users using there mobile application. The plugin generates a javascript code and put it in the head section of website which then get connected to our chat application.

== Installation ==

1. Upload codeteam-livechat directory to the /wp-content/plugins directory
2. Activate the plugin through the Plugins menu in WordPress
3. Enter your Client Id and set other plugin preferences in Admin panel and Save Preferences.
